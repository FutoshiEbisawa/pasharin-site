プライバシーポリシー

最終更新日：2026年5月6日 ／ 適用開始日：2026年5月6日

PashArIn（以下「当方」といいます）は、当方が提供するiOSアプリケーション「パシャりん」（以下「本アプリ」といいます）における利用者の情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

本アプリは、おおむね小学生を主な利用者として想定しています。本ポリシーは、保護者の方が確認しやすいよう、できるだけ平易な日本語で記述するよう努めています。本アプリをご利用いただく際は、保護者の方による確認をお願いいたします。

1. 取得する情報

本アプリは、以下の情報を取得します。取得していない情報は記載していません。

区分内容取得方法
撮影画像 利用者がカメラで撮影した算数の問題（参考書・プリント等）の画像 利用者の操作によりその都度取得
匿名識別子 Supabase Anonymous Authentication が発行する利用者ごとに固有の識別子（UUID）。氏名等の個人情報は含みません アプリ初回起動時に自動取得
チャット入力テキスト 利用者がフォローアップ質問のために入力した文字列 利用者の操作により取得
音声 利用者が音声入力ボタンを押下中の発話 端末内（オンデバイス）で文字に変換し、音声データそのものは外部に送信しません（詳細は第3.3項）
学年情報 利用者または保護者が設定画面で任意に選択した小学校の学年（1〜6） 端末内に保存し、AI に送信される際の難易度ヒントとして利用
アプリ利用状態 オンボーディング完了の有無等の軽微な状態 端末内（UserDefaults）にのみ保存。外部送信なし

本アプリは、以下の情報を 取得しません。

  • 氏名、住所、電話番号、メールアドレス
  • 端末の位置情報（GPS）
  • 連絡先、写真ライブラリ（撮影画像を除く）、カレンダー、ヘルスケアデータ
  • ブラウザ閲覧履歴・他アプリの利用履歴
  • 広告識別子（IDFA）・トラッキング情報
  • 第三者の解析・広告 SDK 経由のデータ
  • App Store 経由の購入情報（本バージョンではアプリ内課金を提供していません）
  • プッシュ通知トークン（本バージョンではプッシュ通知を提供していません）
  • Sign in with Apple 等の外部ログイン情報（本バージョンでは外部ログインを提供していません）

2. 利用目的

取得した情報は、以下の目的にのみ利用します。

  1. 撮影画像、チャット入力テキスト、学年情報を AI（OpenAI 社）に送信し、算数の問題を理解して解説および答えを生成するため
  2. 匿名識別子により利用者ごとのリクエスト回数を管理し、サービスの安定運用・コスト管理を行うため
  3. 不正利用や悪意のある大量リクエストの検知・防止のため
  4. 障害解析・サービス改善のため

これら以外の目的（広告配信、第三者への販売、プロファイリング等）には利用しません。

3. 第三者への提供および利用の委託

本アプリは、サービス提供のため以下の第三者にデータを提供します。

3.1 OpenAI, L.L.C.（米国）

提供データ撮影画像、チャットで入力されたテキスト、学年情報
提供目的画像認識および解説文・回答の生成（gpt-4o / gpt-4o-mini モデル）
通信HTTPS（TLS）による暗号化通信
学習目的での利用OpenAI 社は、API 経由で送信されたデータを同社モデルの学習に利用しないと表明しています

3.2 Supabase Inc.（米国）

提供データ匿名識別子、API リクエストのメタデータ（時刻、回数、エラー有無等）
提供目的認証および API ゲートウェイとしての機能提供
通信HTTPS（TLS）による暗号化通信
参考Supabase Privacy Policy

3.3 Apple Inc.

App Store を通じたアプリ配布のほか、以下の OS 機能を利用しています。

  • 音声認識（SFSpeechRecognizer）：本アプリは音声を 端末内（オンデバイス）でのみ 処理するよう requiresOnDeviceRecognition = true を設定しています。Apple 社のクラウド音声認識サーバーには音声データを送信しません。
  • カメラ・マイクへのアクセス：iOS の標準権限ダイアログを経由してのみ取得します。

当方は、Apple 社が App Store の運営に必要として収集する集計データ（App Analytics 等）の取り扱いを制御できません。詳細は Apple 社のプライバシーポリシー をご確認ください。

4. 国外への移転

上記 OpenAI 社および Supabase 社のサーバーは米国に所在し、当方が提供する情報の処理は米国内で行われます。本アプリをご利用いただくにあたっては、これら米国への情報移転が行われることに同意していただくものとします。

米国は、日本の個人情報保護法令と同水準の保護制度が必ずしもあるわけではありませんが、両社とも業界標準のセキュリティ対策（暗号化通信・アクセス管理・監査等）を講じているとの公表があり、当方もこれに依拠しています。

6. セキュリティ

  • すべての通信は HTTPS（TLS）により暗号化されます。
  • OpenAI 社の API キー等の秘密情報はサーバー側にのみ存在し、アプリ内には埋め込まれていません。
  • 撮影画像はサーバーに永続保存されません。
  • 音声データは端末内でのみ処理され、外部に送信されません。

ただし、インターネット上のデータ通信に絶対的な安全性は存在しません。万が一情報漏洩等の事故が発生した場合は、本ポリシー末尾の問い合わせ先までご連絡ください。

7. 児童（小学生）のプライバシーへの配慮

本アプリの主な利用者は小学生です。当方は以下の方針で児童のプライバシーを保護します。

  • 氏名・住所・学校名等の個人を特定できる情報を 取得しません。
  • 第三者の広告を表示しません。
  • 行動追跡（クロスサイト・クロスアプリトラッキング）を行いません。
  • AI チャットは「直前の算数の問題に関連する話題のみ」に応答するよう設計されています。お子様が個人情報を入力しても、AI は内容を掘り下げたり関連質問を返したりしません。
  • お子様が初めてご利用される際は、保護者の方による設定確認・本ポリシーの確認をお願いします。

8. 利用者・保護者の権利

利用者または保護者の方は、当方に対して以下を請求できます。

  • 取得情報の利用停止
  • サーバー側に残る匿名識別子に紐づくログの削除
  • 本ポリシーに関する説明

請求は本ポリシー末尾の問い合わせ先までご連絡ください。なお、本アプリは氏名・メールアドレス等の連絡情報を取得していないため、削除請求にあたっては該当する匿名識別子（アプリ内「設定」画面で確認可能）をお知らせください。

アプリを削除することで、当該端末からのリクエストは以後送信されなくなります。

9. 本ポリシーの改定

法令の変更、サービス内容の変更、その他の事情により、本ポリシーを改定する場合があります。重要な変更がある場合は、本アプリまたは本ウェブサイト上で告知します。改定後の本ポリシーは、本ウェブサイトに掲載された時点から適用されます。
